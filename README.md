# Smart Study Planner & Focus Tracker

A comprehensive, beautifully designed web application built to help students and professionals manage their tasks, track their study sessions, and maintain productivity streaks.

## 🚀 Features

- **📊 Dashboard Analytics:** Get a quick overview of your total study time, completed tasks, focus sessions, and current daily streak.
- **📝 Advanced Task Management:** Easily add, edit, delete, and search tasks. Assign subjects, estimated durations, and priority levels (High, Medium, Low) to keep your work organized.
- **⏱️ Pomodoro Focus Timer:** Built-in customizable focus timer. Set your preferred focus and break durations to maintain deep work cycles without burnout.
- **📈 Weekly Overview Chart:** Visualize your productivity with an interactive chart tracking your study hours and tasks completed over the last 7 days.
- **🔥 Study Streaks:** Stay motivated with a GitHub-style contribution graph that visually tracks your daily study habits and intensity.
- **🌓 Light & Dark Modes:** A polished, professional UI that fully supports both light and dark themes to suit your preference and reduce eye strain.
- **💡 Daily Motivation:** Start your day right with randomly generated motivational quotes.

## 🛠️ Technology Stack

- **HTML5:** Semantic and accessible structure.
- **CSS3 (Vanilla):** Custom, responsive, and modern styling with CSS variables for seamless theme switching.
- **JavaScript (Vanilla):** Pure, lightweight logic managing state, local storage, timer intervals, and DOM manipulation.
- **Chart.js:** Used for rendering the interactive Weekly Overview analytics chart.
- **FontAwesome:** For clean, scalable vector icons.

## 💻 Installation & Usage

Since this is a purely frontend application, no complex backend setup or installation is required!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ayush-011/smart_study_planner.git
   ```
2. **Open the project:**
   Simply open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge).

## 📂 Project Structure

- `index.html` - The main layout and structure of the application.
- `style.css` - All styling, including light/dark theme variables, responsive grids, and animations.
- `script.js` - Application logic, state management (Local Storage), timer functionality, and chart rendering.
